这是我推荐的Mac应用，持续更新

推荐使用 Homebrew 来安装和管理 MacOS 上的应用(e.g. brew install vim)

我的应用选择标准，重要性依次降低：

  1. 开源应用（可以付费）
  2. 开发者长期维护
  3. 只保留一个同功能的应用
  4. 简洁且漂亮的界面

工具

Homebrew : MacOS 上的包管理器

Zsh : 用于替代Bash的交互式shell

iTerm2: 用于替代Terminal的终端模拟器

Stats: 菜单栏上的状态监控器

Ubersicht : 用于在桌面上显示自定义widgets

ClashX : Clash客户端，用于学习？免费且开源

ClashX Pro : Clash Premium 客户端，ClashX 进阶版本，免费但不开源

Config-for-Clash : 我使用的 Clash Premium 配置，包括分流和轻度广告过滤

Karabiner : 键盘键位修改器

GPG tools : 管理 GPG 密钥

KeeWeb : 基于KeePass的密码管理器

Kdeconnect : 多设备间传送文件和消息同步

Lazygit : Git命令的终端UI

IINA : 媒体播放器

Keka : 高性能文件压缩应用

Hidden Bar : 隐藏菜单栏图标

Free Download Manager : 多线程下载软件

LuLu : MacOS 开源防火墙

HandShaker : 管理和传输 Android 中的文件

文档

Skim : PDF 阅读器

Typora : 所见即所想的 markdown 编辑器

Obsidian : 有非常多插件的双链笔记软件（我同时用来做日程管理）

Cubox : 收集、阅读、管理和回顾网络碎片（我用作Obsidian的上游）

效率

Raycast : Spotlight 和 Alfred 的替代品

Toggl Track : 时间跟踪软件

Hammerspoon : 自动化工具，用于简化操作

Hammerspoon-config : 我使用的 Hammerspoon 配置

Hazel : 自动化工具，主要用在文件管理

Syncthing : 简单安全的多设备文件同步软件

Amethyst : 自动平铺窗口管理器，类似LINUX上的i3wm

Secretive : 用来存储和管理 SSH 密钥的应用，并可以使用指纹快速登陆

编程

NeoVim : 支持更多插件的 vim 编辑器

NeoVim-Config : 基于 Colemak 布局的 Nvim 配置

VS code : 可以通过插件几乎扩展成IDE的编辑器

Ranger: 终端类似 vim 的文件管理器

通讯

Mail: Mac自带的邮件应用，简单又足够用

Telegram : E2EE（需要在加密通讯下）即时通讯应用

Keybase : E2EE 文件传输和通讯（所有情况下）
